# demo-solana

Solana + EVM cross-chain bridge example using `@stableflow/wallet-solana` and RainbowKit. All `@stableflow/*` packages use **beta**.

## Run

```bash
pnpm dev:demo-solana
```

Open **`http://localhost:3005`**.

## Environment

| Variable | Description |
|----------|-------------|
| `VITE_STABLEFLOW_JWT_TOKEN` | Optional API JWT |
| `VITE_WALLET_CONNECT_PROJECT_ID` | WalletConnect for EVM |
