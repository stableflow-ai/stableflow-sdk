# demo-tron

Tron + EVM cross-chain bridge example using TronLink/OKX adapters and `@stableflow/wallet-tron`. All `@stableflow/*` packages use **beta**.

## Run

```bash
pnpm dev:demo-tron
```

Open **`http://localhost:3008`**.

## Environment

| Variable | Description |
|----------|-------------|
| `VITE_STABLEFLOW_JWT_TOKEN` | Optional API JWT |
| `VITE_WALLET_CONNECT_PROJECT_ID` | WalletConnect for EVM |
