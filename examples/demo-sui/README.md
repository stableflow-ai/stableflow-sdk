# demo-sui

Sui + EVM cross-chain bridge example using `@mysten/dapp-kit-react` and `@stableflow/wallet-sui`. All `@stableflow/*` packages use **beta**.

## Run

```bash
pnpm dev:demo-sui
```

Open **`http://localhost:3006`**.

## Environment

| Variable | Description |
|----------|-------------|
| `VITE_STABLEFLOW_JWT_TOKEN` | Optional API JWT |
| `VITE_WALLET_CONNECT_PROJECT_ID` | WalletConnect for EVM |
