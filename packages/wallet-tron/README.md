# @stableflow/wallet-tron

Tron wallet adapter for the StableFlow SDK.

## Installation

```bash
pnpm add @stableflow/wallet-tron @stableflow/core @stableflow/bridges
```

**Peer dependencies:**

- `tronweb` ^6.3.0
- `@layerzerolabs/lz-v2-utilities` ^3.0.151

## When to use

Tron (TRX / USDT) as source or destination in bridge routes.

## Main exports

```ts
import TronWallet, { TronWallet as Named } from '@stableflow/wallet-tron';
```
