# @stableflow/wallet-aptos

Aptos wallet adapter for the StableFlow SDK.

## Installation

```bash
pnpm add @stableflow/wallet-aptos @stableflow/core @stableflow/bridges
```

**Peer dependencies:**

- `@aptos-labs/ts-sdk` ^6.3.1

## When to use

Aptos chain transfers and quotes in `BridgeSFA` flows.

## Main exports

```ts
import AptosWallet, { AptosWallet as Named } from '@stableflow/wallet-aptos';
```
