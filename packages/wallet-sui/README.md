# @stableflow/wallet-sui

Sui wallet adapter for the StableFlow SDK.

## Installation

```bash
pnpm add @stableflow/wallet-sui @stableflow/core @stableflow/bridges
```

**Peer dependencies:**

- `@mysten/sui` ^2.16.2

## When to use

Sui chain transfers and quotes in `BridgeSFA` flows.

## Main exports

```ts
import SuiWallet, { SuiWallet as Named } from '@stableflow/wallet-sui';
```
